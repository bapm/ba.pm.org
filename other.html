Gtk2

07 Apríla 2008 Emmanuel mal krátku prezentáciu o Gtk2 a Perle.

This presentation shows how to build graphical user interfaces through various languages such as C, Perl and even shell scripting (thanks to zenity). The real fun starts when custom made applications start to embed complex components such as Gecko (Mozilla) or VTE (terminal emulator).

Files:

  • color.pl - A simple color picker available in the system tray
  • color-debian.pl - Yet another simple color picker available in the system tray
  • emozilla.pl - A lamentable nonaward-winnning web browser
  • hello-world.pl - Simple hello world in Perl
  • hello-world.c - Simple hello world in C
  • terminal.pl - Run a terminal with custom actions
  • term-top.pl - Run top within a terminal
  • svn-commit.sh - Subversion frontend written in bash using zenity

Presentation in OpenOffice and PDF format.
All files perl-gtk2-presensation.tar.gz
