Xacobeo

je program ktorý pomocou jednoduchého GUI zobrazuje XML a vie spúšťať XPath queries a zobraziť ich výsledok.

Xacobeo poskytuje prostredie aby sa bez problémov a jednoducho dali písať, testovať a spúšťať XPath queries. Vie zobraziť Document Object Model (DOM) a namespaces ktoré sú použité. Program automaticky zaregistruje všetky namespaces a každý element je zobrazený so svojím namespace. Umožňuje rýchly prehľad o XML dokumente.

Xacobeo nieje XML editor, aspoň nie teraz. Jeho účelom je písanie a spúšťanie XPath queries.

Domovská stránka projektu, GnomeFiles, CPAN link

Vychytávky

  • Perl+GTK2+XML::LibXML
  • XML syntax highlighting (XS implementation)
  • XML DOM tree view
  • NS auto registrácia
  • XPath on-type syntax checking
